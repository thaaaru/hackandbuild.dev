Roadmaps and Guides

Contribute a Roadmap

We welcome new roadmaps! To add one, simply drop a new HTML file into the `/roadmaps` folder and update the **`pages.json`** manifest. This keeps the site fast and easy to maintain.

`pages.json` Format

[
    {
        "filename": "ExampleRoadmap.html",
        "title": "A Great Roadmap Example",
        "description": "A brief, one-sentence summary of what this roadmap covers."
    },
    {
        "filename": "AnotherGuide.html",
        "title": "My Second Guide",
        "description": "Focuses on advanced topics in IT."
    }
]

The **`filename`** is required, and **`title`** and **`description`** are highly recommended for the best display.
